#  Tablas de Multiplicar

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base for mult                               [number] [required]
  -l, --list     Show results in console              [boolean] [default: false]
  -e, --end      End for mult                             [number] [default: 10]
```
